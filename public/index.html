Страницы

JSON API (не HTML)

Ссылки ниже открывают сырой JSON в браузере — это нормально.

  • /health — статус сервера и БД
  • /metrics — JSON-метрики
  • /metrics/prometheus — Prometheus
  • /api/time — unix-время
  • /api/notes — список заметок

Примеры curl

curl http://127.0.0.1:8080/health
curl -X POST http://127.0.0.1:8080/api/notes \
  -H "Content-Type: application/json" \
  -d '{"title":"todo","content":"milk"}'
curl -H "X-API-Key: dev-api-key" http://127.0.0.1:8080/api/me
